Suppression des données AdminIA

Vous pouvez supprimer à tout moment certaines ou la totalité de vos données enregistrées dans l'application AdminIA. Les actions ci-dessous sont accessibles directement depuis l'application.

1. Suppression d’un document

Depuis l'application AdminIA :

  1. Ouvrez l'application.
  2. Accédez à Mes Documents.
  3. Sélectionnez le document à supprimer.
  4. Cliquez sur Supprimer.

Toutes les copies du document seront définitivement supprimées.

2. Suppression d’une note

  1. Ouvrez l'application.
  2. Accédez à l’onglet Note.
  3. Sélectionnez la note souhaitée.
  4. Cliquez sur Supprimer.

3. Suppression des résultats d’analyse IA

  1. Accédez à la section Historique IA (selon version).
  2. Sélectionnez l’analyse à supprimer.
  3. Cliquez sur Supprimer.

4. Suppression manuelle via demande

Si vous souhaitez demander la suppression manuelle d’une partie de vos données, vous pouvez envoyer une demande à l'adresse suivante :

support@myadminia.com

(ou l'adresse finale lorsque vous la créerez)

Votre demande sera traitée sous 30 jours conformément au RGPD.
